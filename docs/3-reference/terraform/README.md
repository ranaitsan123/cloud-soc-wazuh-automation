# Terraform Reference

This section is a reference for Terraform-related documentation in the project.

## Existing Terraform docs

- `docs/Terraform/RESOURCE_IMPORT_GUIDE.md`
- `docs/Terraform/RESOURCE_FIXES_GUIDE.md`
- `docs/Terraform/DEPLOYMENT_CHECKLIST.md`
- `docs/Terraform/RESOURCE_MANAGEMENT_ANALYSIS.md`
- `docs/Terraform/CODE_ISSUES_REPORT.md`

## Purpose

These documents explain how Terraform is used to manage AWS infrastructure, how resource imports are handled, and how to validate deployments.

## How to use

1. Review the `RESOURCE_IMPORT_GUIDE.md` for importing existing resources.
2. Use the `DEPLOYMENT_CHECKLIST.md` to validate the deployment steps.
3. Consult `CODE_ISSUES_REPORT.md` for any known implementation gaps.
